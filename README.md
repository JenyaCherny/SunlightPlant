### SunlightPlant inventory storage tool
This an API application is intended to help the user to track an inventory of trees and bushes at SunlightPlant store.
The user can add/post a new arrived inventory item, get or update details on a particular item and delete an item.
Description of functionality: 
SunlightPlant inventory can be tested using Postman tool. 
Please open the attached Postman file to obtain an example of testing inventory item.
